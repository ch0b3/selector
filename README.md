# Selector

# Versions

- Go: 1.18.1
- Serverless framework: 3.16.0

## Quick Start

1. Install modules

```
$ npm install
```

2. Compile function

```
$ make build
```

3. Deploy!

```
$ make deploy
```

## Development at local

1. Fix function

2. Compile function
```
$ make build
```

3. start offline mode
```
$ make offline
```
